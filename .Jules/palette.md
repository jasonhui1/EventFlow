## 2024-05-24 - TabBar Keyboard Accessibility
**Learning:** `title` attributes are insufficient for screen readers in complex dynamic components like file-trees and tab bars; always complement them with explicit, descriptive `aria-label`s, utilizing dynamic data like item names where possible. Furthermore, when creating complex UI components with nested actions (like tabs with close buttons), use a focusable `<div>` with `role="tab"`, `tabIndex={0}`, and an `onKeyDown` handler for `Enter` and `Space` key activation. In the `onKeyDown` handler, ensure you check `e.target === e.currentTarget` to prevent nested interactive elements from incorrectly triggering the container's action.
**Action:** Apply `aria-label`s along with `title` attributes for complex dynamic elements. Use explicit keyboard handlers and role attributes for custom interactive components while preventing event bubbling for nested interactive elements.
