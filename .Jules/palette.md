## 2024-05-24 - Dynamic Tabs Accessibility and ARIA Labels
**Learning:** Complex dynamic components like file-trees and tab bars require more than just `title` attributes for screen readers; they need explicit, descriptive `aria-label`s and proper ARIA roles (`role="tablist"`, `role="tab"`). Furthermore, tabs with nested actions (like close buttons) should use a focusable container (`<div role="tab" tabIndex={0}>`) with an `onKeyDown` handler that explicitly checks `e.target === e.currentTarget` to prevent nested interactive elements from incorrectly triggering the container's action.
**Action:** When implementing or updating tabbed interfaces, ensure proper `role` and `aria-label` attributes are added. For keyboard accessibility, ensure the container handles `Enter` and `Space` events properly and prevents event bubbling from nested actionable items.
